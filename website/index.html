Project Snap

snap project [dir]

Generates a structured project context your AI agent reads instantly. Framework detection, entry points, dependency map — no more re-explaining your codebase.

Token-aware · ~3KB output
📜

Git Snap

snap git [options]

Structured changelog from git history. Groups by type (feat, fix, perf, refactor…), detects breaking changes, filters by date/author/tag. Markdown or JSON.

Conventional commits · Release notes
🔐

Env Guard

snap env [dir]

Scans for hardcoded secrets, missing .env variables, .gitignore compliance, and dangerous files. Catches leaks before you commit.

30+ patterns · Severity levels
🔍

API Snap

snap api [dir]

Discovers all API endpoints in your project. Supports Express, Fastify, Hono, Next.js App Router, NestJS, and more. Never lose track of your routes.

10+ frameworks · Table/JSON output
📦

Dep Snap

snap dep [dir]

Analyses dependencies across package.json, requirements.txt, Cargo.toml, go.mod. Detects outdated, redundant, unused, and missing peer deps.

Multi-language · Offline mode
